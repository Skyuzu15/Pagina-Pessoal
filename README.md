# Portfólio Pessoal - Marlon Vinicius

Este é o início do projeto de portfólio pessoal desenvolvido por Marlon Vinicius.  
Atualmente, **apenas a estrutura HTML foi implementada**. As partes de **estilização (CSS)** e possíveis funcionalidades **dinâmicas (JavaScript)** ainda não foram adicionadas.

---

## ✅ O que já foi feito (HTML)

- **Estrutura base do site**, com semântica clara usando `header`, `section`, `footer`, etc.
- **Tela de abertura** com o nome pessoal exibido em destaque.
- **Conteúdo principal** contendo:
  - Introdução com nome e subtítulo.
  - Seção **Sobre Mim** explicando um pouco da história do desenvolvedor.
  - Seção de **Habilidades**, com ícones e nomes de tecnologias.
  - Seção de **Projetos**, com descrição e link (fictício) para visualização.
  - Seção de **Contato**, com e-mail destacado.
  - **Rodapé** com nome, ano e links para redes sociais (GitHub e LinkedIn).
